# Page Load Experiments

## Dependencies

* [nghttp2](http://nghttp2.org) use `nghttpx` with `-s` flag. The certificate should point to the local machine, `192.168.2.1`.
* [Squid](http://squid-cache.org) Enable SSL Bump.
